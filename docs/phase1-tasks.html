WEEK 1
法的・行政手続き
0/4
✓
就業規則で副業OKか確認する 法務 30分
TMKの就業規則・副業規定を確認。不明点はHRへ問い合わせ。副業届が必要な場合は先に提出。
✓
開業届を税務署へ提出する 法務 1〜2時間
国税庁「個人事業の開業届出書」をe-Tax（マイナンバーカード必要）またはPDF印刷で提出。屋号は任意。事業内容は「経営コンサルティング業」など。
✓
青色申告承認申請書を同時提出する 税務 30分
開業届と同時に提出すると最大65万円控除が受けられる。e-Taxで開業届と一緒に処理すると楽。
✓
事業用の銀行口座・カードを開設する 財務 1時間
個人用と分けることで確定申告がシンプルになる。PayPayビジネス・楽天ビジネスなどネットバンクが手軽。
💡 TIP e-Taxを使えばすべてオンライン完結。マイナンバーカード＋ICカードリーダー（またはスマホ）があればOK。開業届は提出義務があるので最初に必ずやること。
WEEK 2
サービス設計・営業準備
0/5
✓
サービスメニュー・料金を1枚に整理する 営業 2時間
スポット相談・DX戦略・Tableau導入・WS登壇の4メニューを料金付きで1枚にまとめる。提案書の代わりに使える「サービス紹介シート」。
✓
ヒアリングシートのテンプレートを作る 営業 1時間
相談を受けたときにすぐ送れるGoogleフォームまたはNotionページ。「現状の課題・目標・期限・予算・担当者人数」を聞く5〜8問。
✓
業務委託契約書のひな形を用意する 法務 1〜2時間
JNSA等の無料ひな形をベースに「守秘義務・成果物の定義・支払条件・解約条項」を確認。最初の案件前に必ず準備。
✓
請求書テンプレートを用意する 財務 30分
Misocaやfreeeのテンプレートが最速。屋号・振込先・インボイス登録番号（任意）を入れておく。インボイスは事業規模が小さい間は不要なケースも多い。
✓
知人・紹介候補リストを10人書き出す 営業 30分
MBA同期・前職・グロービス人脈など。「AIやDXで困ってそうな人」「中間管理職で変革を任されてそうな人」を優先。連絡のタイミングを決めておく。
WEEK 3
SNS・発信基盤の構築
0/4
✓
Xアカウントを開設・プロフィールを整える SNS 1時間
プロフィールに「生成AI/DX変革コンサル｜トヨタ系DX推進→独立｜MBA」と一言。ヘッダー画像・アイコンも設定。最初の3投稿（自己紹介・専門・実績）を予約投稿。
✓
noteアカウントを開設・初記事を書く SNS 2〜3時間
自己紹介note（2000〜3000字）を最初に投稿。「なぜ独立したか・何が得意か・どんな人を助けたいか」の3構成。Xプロフに固定リンク。
✓
発信テーマ・月間投稿プランを決める SNS 1時間
X：週3本（AITips・DX実体験・考察）。note：月2本（事例・深掘り解説）。テーマを「生成AI活用」「変革マネジメント」「DX失敗あるある」の3軸に絞る。
✓
集客エージェントをセットアップする エージェント 2〜3時間
Claudeのプロジェクト機能またはAPIで、X投稿案・note構成案を自動生成するプロンプトを整備。週のテーマを入力→5投稿案が出る状態にする。
💡 TIP 最初のnoteは「完璧」より「公開」優先。500字の自己紹介でもOK。発信を始めること自体が信頼構築の第一歩。
WEEK 4
マッチングサービス登録・初動
0/4
✓
ビザスクに登録・プロフィールを完成させる マッチング 2時間
スポット相談特化。時給制で1回1〜2時間から受けられる。「生成AI導入・DX変革」の専門家として登録。TMKでの実績を具体的に書くと案件が来やすい。
✓
Workshipに登録する（副業案件向け） マッチング 1時間
プロジェクト型・週1〜2日の副業案件が多い。DX推進・AI活用のPM・アドバイザー案件を探す。スキルタグ「DX・生成AI・Tableau・組織変革」を設定。
✓
知人リストへ営業DMを送り始める 営業 1〜2時間
「副業でコンサルを始めました。AI/DXで困っている知り合いがいたら紹介してください」という軽いメッセージ。重くならないカジュアルな一言がコツ。
✓
地域のDX/AI勉強会を1つ見つけて登録する コミュニティ 30分
Connpass・Doorkeeper等で北九州・福岡のAI/DX系イベントを検索。参加→交流→登壇の順番で信頼構築。まず参加するだけでOK。

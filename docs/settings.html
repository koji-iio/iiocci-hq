⚙️ 初期設定

Google Apps ScriptのURLを登録することで、スマホ・PCどちらからでもデータが同期されます。
一度設定すれば次回から不要です。

Step 1 — Google Apps Scriptをセットアップ
Googleスプレッドシートをデータベースとして使うためのAPIを作成します。
1
Google スプレッドシートを新規作成（タイトルは「IIOCCI HQ データ」など）
2
メニューから 拡張機能 → Apps Script を開く
3
エディタに以下のコードを全て貼り付けて保存（Ctrl+S）
// GASコードを読み込み中...
4
デプロイ → 新しいデプロイ → 種類：「ウェブアプリ」→ アクセス：全員 → デプロイ
5
表示された ウェブアプリのURL をコピーして下に貼り付ける
Step 2 — GAS URLを登録
Step 3 — GitHub Pagesにデプロイ（任意）
URLでどこからでもアクセスできるようにする手順です。
1
GitHubで新しいリポジトリを作成（例：iiocci-hq）、Publicに設定
2
このフォルダ内の全HTMLファイルをリポジトリにアップロード（GitHub Desktopが簡単）
3
Settings → Pages → Source: main branch / docs folder → Save
4
https://[ユーザー名].github.io/iiocci-hq/ にアクセスできるようになる
5
このURLをスマホのホーム画面に追加すれば完成 🎉
